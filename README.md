# Better Craft Bonus

Better Craft Bonus is a mod that allows you to change the craft bonus when you create items in a crafting station (forges, cauldron, modded stations, etc).

![](https://i.imgur.com/jJErHli.png)

Vanilla gives you one shot per item crafted. If it is a pack of items (i.e: Salad x3) the game rolls its dice 3 items and may give you 3, 4, 5 or 6. Besides, it always shows "+1" at the bonus text.

This mod will change that rule:
* By default, once installed it will always multiply by 2 the recipe when you get the bonus.
  * Example: if you craft a saladx3 and get the bonus, you will receive 3x2=6 fixed salads all times.
* If you use shift for multicrafting (x5 in vanilla), this will be multiplied by that multicrafting game amount.
  * Example: if you multicraft a saladx3 (x15) and get the bonus, you will receive 15x2=30 salads all times.
* The bonus text will be updated accordingly, showing "x2" by default.

IMPORTANT: If you run out of space in your inventory, the remaining bonus items that didn't fit will drop on the ground. Don't forget to collect them!

# Configuration

Configuration is available in the .cfg file to customize the multiplier if you want to make it different. You can customize next values:

![](https://i.imgur.com/uWj3kdU.png)

* craft bonus amount → Multiplier to apply when the bonus is triggered (default = x2)
* craft bonus chance → Probability of craft bonus to happen against your current skill level (default = 0.25, which turns into 25% chance at skill = 100)
* multi craft amount → Amount to craft when Shift is pressed for multicrafting in craft stations (default = 5 as in vanilla)
